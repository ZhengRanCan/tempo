# content-rules.md

## 目的

本文档定义小程序中的全局文案风格、AI 陪伴表达、错误提示和仪式感表达边界。

页面专属文案目标以 `docs/design/page/pages/` 为准。本文档只约束跨页面一致的表达原则。

## 文案基调

整体文案应呈现：

- 温和。
- 具体。
- 可执行。
- 不评判。
- 不夸大。

产品语气像一个可靠的任务伙伴，而不是老师、审判者或算命工具。

## 今日任务文案

今日任务必须清楚表达：

- 今天做什么。
- 最低完成线是什么。
- 为什么先做这一步。
- 建议什么时候做。

推荐：

```text
今天先完成开题报告的研究问题草稿。
最低完成线：写出 3 个候选研究问题，不需要一次写完。
```

避免：

```text
今天必须全力以赴，否则你会落后。
宇宙能量显示你今天适合完成全部任务。
```

## 最低完成线文案

最低完成线必须具体、低压力、可执行。

推荐：

```text
至少写出 3 个要点
至少读完 1 篇文献摘要
至少整理出 5 个关键词
至少完成 25 分钟专注
```

避免：

```text
尽量多做一点
保持努力
进入高效状态
完成全部任务
```

## low 状态文案

`low` 状态下，文案应降低压力。

推荐：

```text
今天先保留一点推进感就好。
最低完成线完成后，就算今天已经往前走了一步。
```

避免：

```text
你今天状态不好，需要更加努力。
低能量会影响你的目标完成。
```

## 复盘文案

复盘不应评判用户。

推荐：

```text
今天哪些任务有推进？
哪些任务只完成了一部分？
明天我会根据今天的情况帮你调整。
```

避免：

```text
你今天为什么没有完成？
你已经落后计划。
```

## AI 陪伴表达

允许：

- 温和提醒。
- 简短鼓励。
- 今日关键词。
- 轻量仪式感表达。
- 对任务安排的解释。

不允许：

- 健康诊断。
- 心理诊断。
- 命运预测。
- 对 MBTI、星座、时辰、塔罗做科学化解释。
- 用空泛鼓励替代任务建议。

## 塔罗、MBTI、每日关键词边界

这些元素只能作为表达风格和仪式感元素。

允许：

```text
今天的关键词：启动
如果你偏好晚上专注，可以把最重要的一步放到晚饭后。
```

不允许：

```text
你的 MBTI 决定了你今天不适合沟通。
塔罗显示这个任务今天一定会成功。
这个时辰决定了任务成功率。
```

## 错误文案

错误文案必须包含：

- 问题是什么。
- 用户可以怎么做。

推荐：

```text
截止日期不能早于今天，请重新选择一个日期。
每日可用时间需要大于 0 分钟。
计划暂时生成失败，已为你使用基础规则生成安排。
```

避免：

```text
发生错误。
参数非法。
请求失败。
```

## 空状态文案

空状态必须提供下一步动作。

推荐：

```text
还没有目标
先写下一个想推进的目标，我来帮你拆成每天能做的小步。
```

避免：

```text
暂无数据
没有内容
你还没有任何任务
```

## 按钮文案

按钮必须使用明确动作。

推荐：

- 创建目标。
- 生成计划。
- 开始今日任务。
- 保存复盘。
- 调整计划。
- 查看今日任务。

避免：

- 确定。
- 提交。
- 下一步。
- 好的。
- 继续。

如果使用「下一步」，上下文必须非常明确。

## 禁止表达

默认禁止出现以下表达：

- 你失败了。
- 你落后了。
- 你必须完成。
- 你的状态很差。
- 系统预测你今天会成功。
- 星座决定了你的效率。
- MBTI 说明你不适合做这件事。
- 塔罗决定今天的任务安排。
- 你应该更自律。

## 文案优先级

文案优先服务任务执行。

优先级：

1. 说明下一步动作。
2. 降低启动成本。
3. 解释任务安排。
4. 提供温和陪伴。
5. 提供轻量仪式感。

不要让仪式感文案压过任务本身。
