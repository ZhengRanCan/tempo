# visual-system.md

## 目的

本文档定义小程序的全局视觉 token，包括颜色、字体、字号、间距、圆角和视觉层级。

本文档不定义页面结构，也不定义某个页面的组件顺序。页面级结构以 `docs/design/page/` 为准；按钮、卡片、标签、表单等组件规则以 `docs/design/components.md` 为准。

## 视觉目标

- 温和：降低任务管理带来的压力。
- 清晰：让用户快速知道当前能做什么。
- 可执行：视觉层级服务于任务推进，不制造装饰负担。
- 克制：AI、塔罗、MBTI、每日关键词只能作为轻量表达元素，不能压过任务本身。

## 颜色系统

颜色必须先表达语义，再考虑装饰效果。

### 基础色

| Token | 色值 | 用途 |
|---|---|---|
| `canvas` | `#FAF8F3` | 页面默认背景 |
| `surface` | `#FFFFFF` | 卡片、表单、弹窗、主要内容块 |
| `surface-soft` | `#F3EFE7` | 次级信息块、日期块、弱背景 |
| `surface-muted` | `#ECE6DA` | 禁用态、弱强调背景 |
| `ink` | `#24211C` | 主标题和重要文字 |
| `body` | `#4B463D` | 正文 |
| `muted` | `#7C7568` | 辅助文字 |
| `border` | `#E5DED2` | 分割线和卡片边界 |

### 强调色

| Token | 色值 | 用途 |
|---|---|---|
| `accent` | `#6B6FD6` | 主按钮、当前日期、关键选中态 |
| `accent-soft` | `#ECECFF` | 轻强调背景 |
| `accent-pressed` | `#555AC0` | 主按钮按下态 |
| `warm` | `#D68A5A` | 今日关键词、轻仪式感标签 |
| `warm-soft` | `#FFF2E8` | 温和提示背景 |

### 状态色

| Token | 色值 | 用途 |
|---|---|---|
| `success` | `#4F9D69` | 已完成 |
| `success-soft` | `#EAF6EE` | 已完成背景 |
| `warning` | `#D7A245` | 部分完成、需要调整 |
| `warning-soft` | `#FFF5DA` | 部分完成背景 |
| `danger` | `#C75C54` | 严重错误，少量使用 |
| `danger-soft` | `#FBEDEA` | 错误背景 |
| `neutral` | `#9A9488` | 未开始、跳过、低优先级 |

## 颜色使用规则

- 页面背景使用 `canvas`。
- 卡片、表单、弹窗使用 `surface`。
- 次级区域使用 `surface-soft`。
- 主按钮、关键选中态和当前日期使用 `accent`。
- 普通未完成、跳过、低能量不使用 `danger`。
- `warm` 只用于轻仪式感元素，不用于核心完成状态。
- 单个页面中强调色面积不应超过 20%。
- 颜色不能作为唯一状态信号，必须搭配文字或图标。

## 字体

小程序优先使用系统字体：

```text
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif;
```

## 字号层级

| Token | 字号 | 字重 | 行高 | 用途 |
|---|---:|---:|---:|---|
| `display` | 28px | 600 | 1.25 | 仅用于特别重要的首屏核心信息 |
| `title-lg` | 22px | 600 | 1.3 | 页面标题或重点卡片标题 |
| `title-md` | 18px | 600 | 1.35 | 卡片标题 |
| `body` | 16px | 400 | 1.55 | 正文、任务说明 |
| `body-sm` | 14px | 400 | 1.5 | 辅助说明 |
| `caption` | 12px | 500 | 1.4 | 标签、状态、时间 |

## 字体规则

- 单个页面最多使用 3 个字号层级。
- 页面主体不要重复展示顶部导航栏已经表达的页面大标题。
- 标题不使用过度粗体。
- 不使用全大写英文。
- 重要信息优先靠位置、留白和组件层级表达，不靠反复加粗。
- 长文案必须拆短，适合移动端快速扫读。

## 间距

| Token | 值 | 用途 |
|---|---:|---|
| `xs` | 4px | 图标与文字间距 |
| `sm` | 8px | 紧凑元素间距 |
| `md` | 12px | 表单和控件内部间距 |
| `lg` | 16px | 卡片内部间距 |
| `xl` | 24px | 页面区块间距 |
| `xxl` | 32px | 大区块间距，谨慎使用 |

## 间距规则

- 页面主体顶部间距以 `docs/design/page/01-page-layout-contract.md` 为准。
- 卡片内部优先使用 `lg`。
- 表单字段之间优先使用 `md` 或 `lg`。
- 不使用大块顶部空白制造页面标题区。
- 不用过密分割线代替空间层级。

## 圆角

| Token | 值 | 用途 |
|---|---:|---|
| `sm` | 6px | 小标签 |
| `md` | 10px | 输入框、按钮 |
| `lg` | 16px | 卡片、日期块 |
| `xl` | 24px | 重点卡片、弹窗 |
| `pill` | 999px | 状态标签、胶囊选择 |

## 视觉层级

- 层级主要通过背景色、边框、留白和字号表达。
- 默认不使用强阴影。
- 卡片使用 `surface + border`。
- 次级区域使用 `surface-soft`。
- 重点区域可以小面积使用 `accent-soft` 或 `warm-soft`。
- 不使用大面积渐变、玻璃拟态、强投影或高饱和背景。

## 禁止项

- 禁止大面积渐变。
- 禁止玻璃拟态。
- 禁止强阴影。
- 禁止用红色表达普通未完成、跳过或低能量。
- 禁止把星座、MBTI、塔罗、时辰作为主视觉背景。
- 禁止在页面主体重复顶部导航栏已经表达的页面标题。
