# visual-system.md

## 目的

本文档定义本项目的视觉系统，包括色彩、字体、间距、圆角和基础视觉规则。

本项目视觉目标是温和、低压力、清晰、可执行。

---

## 色彩系统

色彩语义优先，不做装饰性堆叠。

### 基础色

| Token | 色值 | 用途 |
|---|---|---|
| canvas | #FAF8F3 | 页面默认背景，暖白色 |
| surface | #FFFFFF | 卡片、输入区域 |
| surface-soft | #F3EFE7 | 次级信息块、日期块背景 |
| surface-muted | #ECE6DA | 轻强调背景 |
| ink | #24211C | 主标题和重要文字 |
| body | #4B463D | 正文 |
| muted | #7C7568 | 辅助文字 |
| border | #E5DED2 | 分割线和卡片边界 |

|---|---|---|

| warm-soft | #FFF2E8 | 温和提示背景 |

---

## 色彩使用规则

- 页面背景使用 canvas。
- surface 用于任务卡片、表单和弹窗。
- accent 只用于主要行动、当前日期和关键选中态。
- danger 不用于普通未完成任务，避免制造失败感。
- warm 只用于轻仪式感元素，不用于核心完成状态。
- 单个页面中强调色面积不应超过 20%。
- 颜色不能作为唯一状态信号，必须搭配文字或图标。

## 字体

小程序优先使用系统字体。

```text
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif
```

---

## 字号层级

| Token | 字号 | 字重 | 行高 | 用途 |
|---|---:|---:|---:|---|
| display | 28px | 600 | 1.25 | 首页或今日主标题 |
| title-lg | 22px | 600 | 1.3 | 页面标题 |
| title-md | 18px | 600 | 1.35 | 卡片标题 |
| body | 16px | 400 | 1.55 | 正文、任务说明 |
| body-sm | 14px | 400 | 1.5 | 辅助说明 |
| caption | 12px | 500 | 1.4 | 标签、状态、时间 |

---

## 字体规则

- 单个页面最多使用 3 个字号层级。
- 标题不使用过度粗体。
- 不使用全大写英文。
- 重要信息优先用位置和留白强调，不靠加粗堆叠。
- 长文案必须拆短，适合移动端快速扫读。

## 圆角

| Token | 值 | 用途 |
|---|---:|---|
| sm | 6px | 小标签 |
| md | 10px | 输入框、按钮 |
| lg | 16px | 任务卡片、日期块 |
| xl | 24px | 今日重点卡、弹窗 |
| pill | 999px | 状态标签、胶囊按钮 |

---

## 层级表达

- 不使用大面积阴影制造层级。
- 层级主要通过背景色、边框和留白表达。
- 卡片使用 surface + border。
- 次级区域使用 surface-soft。
- 重点区域使用 accent-soft 或 warm-soft，但面积要克制。

---

## 禁止事项

- 禁止使用大面积渐变。
- 禁止使用玻璃拟态。
- 禁止使用强阴影。
- 禁止使用高饱和红色表达普通未完成。
- 禁止把星座、MBTI、时辰作为主视觉背景。
