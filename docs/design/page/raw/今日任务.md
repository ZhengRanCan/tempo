请对微信小程序的「今日任务」页做第二轮 UI 和交互优化。本轮讨论建立在一个前提上：页面内部原来的冗余标题区已经被删除，顶部导航栏已经负责显示当前页名称「今日任务」，底部 TabBar 已经负责页面切换。

本轮目标：
把「今日任务」页改成一个更适合执行的页面，而不是信息展示页。用户打开页面后，应该优先看到“今天最值得先做的一件事”，同时能知道今天是否还有其他任务，并可以进入「今日全部任务」查看完整列表。

产品背景：
这个小程序是一个个性化 AI 任务日历。核心闭环是：用户创建目标 → 系统生成每日任务 → 用户查看今日任务 → 用户输入当天状态 → 系统给出今日建议 → 晚上复盘 → 系统重排后续计划。今日页应该服务这个闭环，不要做成复杂首页。PRODUCT_SPEC.md 里也说明，MBTI、塔罗、每日关键词等只作为表达风格和仪式感包装，不作为严肃预测依据。:contentReference[oaicite:0]{index=0}

一、今日页整体结构

请将「今日任务」页整理成以下结构：

1. 顶部导航栏
   - 保留现有顶部导航栏。
   - 中间显示「今日任务」。
   - 不要在页面主体里再出现「今日任务」大标题、页面胶囊标签、页面说明副标题。

2. 今日重点任务卡
   - 页面主体第一张卡必须是「今日重点任务」。
   - 这张卡只展示 AI 或计划逻辑判断出的今日最优先任务，也就是主任务。
   - 不要把所有任务都直接堆在今日页首页。

3. 今日状态调整卡
   - 放在重点任务卡下面。
   - 用于快速调整当天状态。
   - 右上角增加「管理」入口，用来管理常用状态选项。

4. AI 今日建议卡
   - 放在状态卡下面。
   - 默认保持短内容，不要大段解释。
   - 可以包含今日关键词或塔罗灵感，但必须轻量展示。

5. 晚间复盘入口
   - 放在页面下方。
   - 设计成较轻的入口，不要抢占主任务卡的注意力。
   - 后续可以根据时间或任务完成状态条件展示，但本轮先做成轻量卡片即可。

二、今日重点任务卡设计

今日重点任务卡需要包含以下信息：

- 小标签：今日重点任务
- 当前目标名称，例如：完成开题报告初稿
- 说明文案：当前最值得推进的一步
- 今日主任务标题，例如：整理 3 篇核心文献摘要
- 元信息：预计 30 分钟｜中优先级
- 最低完成线，例如：最低完成线：先写出 3 条摘要要点即可
- 主按钮：开始专注
- 次操作：标记完成

重点新增：
如果今天有多个任务，请在主任务卡中加入一个明显但不喧宾夺主的入口：

「今日共 N 个任务 · 查看全部」

这个入口应该放在主任务卡内部，位于最低完成线下方、主按钮上方比较合适。它应该看起来可点击，可以使用浅紫色边框行、右箭头或小列表图标。

如果 todayTasks.length > 1：
- 显示「今日共 N 个任务 · 查看全部」
- 可以附带一行很短的预览，例如：
  其余任务：整理研究背景、补充参考文献、确认研究问题
- 点击后进入「今日全部任务」页面，或打开一个底部抽屉展示全部任务。

如果 todayTasks.length === 1：
- 可以不显示「查看全部」入口，或者显示「今日共 1 个任务」但不要突出。

如果 todayTasks.length === 0：
- 保留当前空状态卡片，引导用户创建目标。
- 不显示今日重点任务卡。

三、今日全部任务页 / 弹层

请新增或完善一个「今日全部任务」入口。

优先方案：
新增一个二级页，例如 pages/today-all 或项目中已有的合适页面，用于展示今天所有任务。

如果当前项目路由结构不方便新增页面，可以先实现成底部抽屉或弹窗，但需要在回传说明中解释原因。

「今日全部任务」页建议结构：

1. 顶部导航栏：今日全部任务
2. 顶部摘要：
   - 今日共 N 个任务
   - 已完成 X 个
   - 预计总时长 X 分钟
3. 任务列表：
   - 第一组：今日重点
   - 第二组：其他任务
   - 可选第三组：低能量备用任务
4. 每个任务项至少展示：
   - 任务标题
   - 所属目标
   - 预计时长
   - 优先级
   - 状态：未开始 / 进行中 / 已完成
   - 最低完成线
   - 操作：开始 / 完成

注意：
今日页首页不要直接展示完整任务列表，只展示主任务和「查看全部」入口。完整列表放到二级页或弹层里。

四、今日状态调整卡

当前状态卡不要做成复杂问卷。首页只保留常用快速选项，但要增加「管理」入口。

卡片标题：
调整今日状态

右上角：
管理 >

卡片内容：

1. 当前能量
   - 低
   - 普通
   - 高
   - 更多
   - 默认选中：普通

2. 可用时间
   - 15 分钟
   - 30 分钟
   - 60 分钟
   - 自定义
   - 默认选中：30 分钟

3. 当前状态
   - 有点焦虑
   - 正常
   - 想冲刺
   - 默认选中：正常

交互要求：
- 点击选项后更新当前选择状态。
- 「自定义」允许用户输入今天可用分钟数。
- 「更多」或「管理」用于进入状态选项管理，不要把所有可选项都铺在今日页里。
- 今日页只显示常用选项，复杂配置放在管理页面、弹窗或「我的-任务偏好」中。

六、AI 今日建议卡

建议结构：

- 其余任务可在状态允许时继续推进。

底部说明：
塔罗灵感仅用于提醒节奏，不作为预测依据。

注意：
