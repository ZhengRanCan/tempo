# F26 Reference Images

本 feature 不重复复制参考图，默认引用：

- 任务日历页：`docs/design/page/reference_image/日历.png`

使用规则：

- 参考图用于硬性判断日历页首屏重心、计划板气质、日期/任务分区、按钮层级和信息密度。
- 不要求逐像素还原，但完成效果必须明显接近参考图；如果只是轻微样式调整，不能通过验收。
- 如果参考图和 `feature.md` 冲突，以 `feature.md` 为准。
- 如果参考图和 `docs/design/page/pages/calendar-page.md` 冲突，标记为待确认，不要自行扩大功能范围。
