## 3. Severity Rule

严重程度：

- P0：阻断主流程，用户无法继续完成核心任务。
- P1：核心功能可继续，但数据、状态或跳转明显错误。
- P2：影响体验或理解，但不阻断主流程。
- P3：文案、视觉、轻微交互问题。
