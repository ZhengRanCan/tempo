# F27 Reference Image

F27 uses the existing project reference image:

```text
docs/design/page/reference_image/创建目标.jpeg
```

This feature intentionally uses fewer written design details than F25/F26. The reference image is the main visual source.

Use the image to judge:

- Whether the page looks like a goal creation flow.
- Whether the first screen starts with core inputs instead of a generic settings form.
- Whether the step cards, spacing, selected chips, optional notes, preference panel, and primary button are visually close to the reference.

Do not require pixel-perfect restoration. Do not pass F27 if only color, radius, or shadow changed while the page structure remains close to the old form.

If the image and `feature.md` conflict, follow `feature.md`. If the image and `docs/design/page/pages/create-goal-page.md` conflict, record the conflict in `docs/progress.md` instead of expanding scope.
